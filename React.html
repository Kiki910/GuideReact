REACT

Cos'è React?

React è una libreria JavaScript per la creazione di interfacce utente, ossia per la creazione di SPA(single Page Application). React venne sviluppato all'interno di Facebook nel 2013, ma ora si è estesa molto ed è una libreria open-source sostenuta da molti programmatori.
La funzionalità principale è lo sviluppo di applicazioni dinamiche che non hanno bisogno di ricaricare la pagina; molto importante è anche la possibilità di vedere le modifiche effettuate sul codice in tempo reale in modo da avere uno sviluppo flessibile, ma soprattutto più rapido.

Installazione
Per poter iniziare a lavorare con React, occorre installare npm e nodejs. Per evitare problemi con l'ultima versione, utilizziamo direttamente i seguenti comandi da terminale: 

                        
    Comando 1)     sudo npm install -g n

    Comando 2)     sudo n stable

    Comando 3)     creiamo il nostro progetto
Creazione a avvio progetto
Per creare un progetto occorre scrivere questo comando nel terminale: 

                        
    Comando creazione progetto -->     npx create-react-app nomeprogetto
In questo modo, nella directory in cui noi abbiamo aperto il terminale, abbiamo una cartella nuova del nostro progetto.
All'interno di questa cartella, abbiamo 3 sotto-cartelle, che sono:
  • node_modules: contiene i moduli che possono essere locali o globali;
  • public: contiene alcuni file pubblici e non si deve toccare;
  • src: contiene i file principali della nostra applicazione, come ad esempio App.js, App.css, index.js...
Oltre a queste cartelle, abbiamo anche dei file. Tra questi, il più importante è il seguente:
  • package.json: contiene dati importanti relativi al progetto, come nome, versione, dipendenze che installiamo e gli scripts che servono per far partire il progetto.

Ora ci occorre solo più far partire il progetto. Per fare questo, abbiamo bisogno del seguente comando: 

                        
    Comando avvio progetto -->     npm start
Attraverso questo comando, non facciamo altro che aprire la pagina nel browser.

NOTA BENE!

Se accidentalmente si rimuove la cartella node_modules, il comando per rimettere le cose a posto è il seguente: 
                                
    Comando -->     npm install

Visual Studio Code

Se si utilizza Visual Studio Code, molto utile è l'estensione per l'aiuto nell'utilizzo di React. La seguente estensione è: 
                                
    Estensione -->     ES7+ React/Redux/React-Native snippets
Sempre se si utilizza Visual Studio Code per scrivere i codici in React, molto utile per l'aiuto nella scrittura del codice, occorre modificare delle preferenze nelle impostazioni: 
                                
    Percorso -->     File / Preferences / Settings / Icona in alto a destra (impostazioni JSON)   

    Codice -->      {
                        "emmet.includeLanguages": {
                            "javascript": "html"
                        }
                    }

PICCOLO AIUTO!

Per aiutarti ad inizio scrittura del codice di un file in React, è possibile scrivere la funzione base in due modi.

Per scrivere la funzione base con la arrow [ => ], bisogn ascrivere il seguente comando: 
                                    
        Comando -->        rafce

        Risultato -->      import React from'react'
                           
                           const App = () => { 
                               return (
                                   <div>App<div>
                               )    
                           } 

                           export defaultApp
Per scrivere la funzione base con la function, bisogn ascrivere il seguente comando: 
                                    
        Comando -->        rfce

        Risultato -->      import React from'react'
                           
                           function App() { 
                               return (
                                   <div>App<div>
                               )    
                           } 

                           export defaultApp
Operatore spred()
Molto spesso, associato alla funzione map() spiegata nella sezione JS Avanzato , viene utilizzato l'operatore spread. Questo operatore è facilmente individuabile attraverso l'utilizzo dei 3 punti [ ... ]. Questo operatore serve per fare una copia perfetta di un array. 

                        
        import React, { useState } from 'react'
        import "./App.css"
        
        const URL = "https://raw.githubusercontent.com/itisrivoira/obelix/main/lista_locali.json"

        const App = () => {
            const[locali setLocali] =  useState([])

            const connessione=()) => {
                fetch(URL)
                .then((LocaliText) => LocaliText.json())
                .then((LocaliJSON) => {
                    LocaliJSON.map((elemento, indice) => {
                        setLocali((locali) => [...locali, <div>{elemento.citta   }</div>
                    })
                })
            ]

            return (
                <div>
                    <div className="container">
                        <div className="button" onClick={connessione}>CONNESSIONE JSON</div>
                    </div>
                    {locali}
                </div>
            )
        }

        export default App
Gli hooks
Gli hooks ci permettono di agganciarci alle funzionalità di React come lo stato e il ciclo.

useState()
Lo useState() è un hook che ci permettere di modificare lo stato di funzione in un componente; lo stato si riferisce a dati o proprietà che servono nell'applicazione.

Per utilizzarlo occorre innanzitutto importarlo. Per fare ciò dobbiamo scrivere questo: 
                                
        import React, { useState } from 'react'

Invece, per "dichiararlo" si deve scrivere questo: 
                                
        const [locali, setLocali] = useState([])

Per stampare il contenuto dell'array, ossia il primo parametro, e per modificare lo stato dobbiamo fare così: 
                                
        Per stampare nel return -->    {locali}

        Per aggiornare lo stato -->    setLocali((locali) => [contenuto per aggiornare lo stato])
Esempio dell'utilizzo dello useState()
                        
        import React, { useState } from 'react'
        import "./App.css"
        
        const URL = "https://raw.githubusercontent.com/itisrivoira/obelix/main/lista_locali.json"

        const App = () => {
            const[locali setLocali] =  useState([])

            const connessione=() => {
                fetch(URL)
                .then((LocaliText) => LocaliText.json())
                .then((LocaliJSON) => {
                    LocaliJSON.map((elemento, indice) => {
                        setLocali((locali) => [...locali, <div>{elemento.citta   }</div>
                    })
                })
            ]

            return (
                <div>
                    <div className="container">
                        <div className="button" onClick={connessione}>CONNESSIONE JSON</div>
                    </div>
                    {locali}
                </div>
            )
        }

        export default App


useEffect()
Lo useEffect() è un hook che ci permettere di sincronizzare un componente cin un sistema esterno. In poche parole possiamo utilizzare lo useEffect() per richiamare una sola volta una funzione all'interno dell'applicazione.

Per utilizzarlo occorre innanzitutto importarlo. Per fare ciò dobbiamo scrivere questo: 
                                
        import React, { useEffect } from 'react'

Invece, per "dichiararlo" si deve scrivere questo: 
                                
        useEffect(() => {
            first

            return ()() => {
                second
            }
        }, [third])

        Per semplicità si utilizza in questo modo:
        
        useEffect(() => {
            funzione()
        }, [])

Esempio dell'utilizzo dello useEffect()
                        
        import React, { useState, useEffect } from 'react'
        import "./App.css"
        
        const URL = "https://raw.githubusercontent.com/itisrivoira/obelix/main/lista_locali.json"

        const App = () => {
            const[locali setLocali] =  useState([])

            useEffect(() => {
                connessione()
            }, [])

            const connessione=() => {
                fetch(URL)
                .then((LocaliText) => LocaliText.json())
                .then((LocaliJSON) => {
                    LocaliJSON.map((elemento, indice) => {
                        setLocali((locali) => [...locali, <div>{elemento.citta   }</div>
                    })
                })
            ]

            return (
                <div>
                    <div className="container">
                        <div className="button" onClick={connessione}>CONNESSIONE JSON</div>
                    </div>
                    {locali}
                </div>
            )
        }

        export default App

useRef()
Lo useRef() è un hook che permette di modificare delle caratteristiche di elementi di React. Viene utilizzato al posto del querySelector di JavaScript.
Per utilizzarlo occorre prima importarlo e lo facciamo nel seguente modo: 
                                
        import React, { useRef } from 'react'

Invece, per "dichiararlo" si deve scrivere questo: 
                                
        const nome = useRef()

Passaggio parametri
Quando si lavora in React, ci vieni molto più semplice suddividere l'applicazione in componenti in modo da avere un codice più semplice e più leggibile. 

                                
    
  App.js

    import React from 'react'
    import "./App.css"
    import Header from "./Header"
    
    const App = () => {

        let mieiParametri={
            logo:{
                pathLogo:https://upload.wikimedia.org/wikipedia/commons/thumb/8/8c/Coat_of_arms_of_Saluzzo.svg/1200px-Coat_of_arms_of_Saluzzo.svg.png,
                percorso:https://comune.saluzzo.cn.it/
            },
            link1:{
                url:https://comune.saluzzo.cn.it/servizio/biblioteca/,
                anchor:BIBLIOTECA
            },
            link2:{
                url:https://comune.saluzzo.cn.it/servizio/scuole/,
                anchor:SCUOLE
            },
            link3:{
                url:https://comune.saluzzo.cn.it/servizio/turismo/,
                anchor:TURISMO
            },
            link4:{
                url:https://comune.saluzzo.cn.it/servizi/tutti-i-servizi-in-ordine-alfabetico/,
                anchor:SERVIZI
            }
        }
        return (
            <div>
                <Header valori={mieiParametri}/>
            </div>
        )
    }

    export default App

React Router Dom
React-router-dom è un modulo non di React, cioè, come si dice, non è nativo, che ci permette di passare da un componente all'altro.
Per poterno utilizzare, dato che non è nativo di React, occorre prima installarlo all'interno della cartella in cui vogliamo creare il nostro progetto; per farlo dovremo farlo nel seguente modo: 

                        
    npm install react-router-dom
