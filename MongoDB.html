Cos'è Mongo DB?

MongoDB è un sistema di gestione di database non relazionale (oppure NoSQL), ossia quei database utilizzati per modelli con schemi flessibili per creare applicazioni moderne. E' in grado di elaborare dati strutturati, semi-strutturati e non, orientato ai documenti e a un linguaggio di query non strutturato.
Utilizza un formato di storage chiamato BSON che è una forma binaria di JSON in grado di memorizzare più tipi di dati.




-Funzionamento Mongo DB

Mongo DB memorizza gli oggetti di dati in raccolte e documenti, al contrario dei database relazionali che raccolgono i dati in tabelle e righe. Le raccolte comprendono insiemi di documenti. I documenti sono formati da coppie chiave-valore che sono l'unità base dei dati in Mongo DB.

La struttura di un documento può essere modificata aggiungengo nuovi campi o eliminandone altri. Le chiavi vengono definite dai documenti, mentre i valori vengono definiti da tipi di dati, compresi altri documenti, array e array di documenti.




-Ricerca testuale

Una funzionalità molto importante in Mongo DB è la ricerca testuale che permette di eseguire query in campi stringa per trovare testo o parole specifiche. Una ricerca di testo può essere eseguita con un indice di testo o opertatore $text.
Un indice di testo può essere una stringa oppure un array di elementi e per eseguire la ricerca occorre che la raccolta contenga un indice di testo. Una raccolta può avere un solo indice di testo e più campi possono avere lo stesso indice di testo.




Installazione

Per poter utilizzare Mongo DB occorre installarlo e abbiamo due strade.
Possiamo installare la Mongo DB Shell oppure installare Mongo DB Compass.
In ogni caso, installare Mongo DB Community Server [download].

Una volta scaricato il file, fare doppio clic ed eseguire il file.




1)
Dalla finestra che si apre in seguito all'Esegui, dovremmo semplicemente cliccare Next.
2)
In seguito, dovremo accettare i termini della licenza e cliccare nuovamente su Next.



3)
In seguito all'eccettazione dei termini, ci verrà chiesto di scegliere la tipologia che abbiamo bisogno. Con la Complete, verranno installate tutte le funzionalità del programma, mentre la Costum consente agli utenti di scegliere quali funzionalità del programma verranno installate.
Ti consiglio di installare la Completa per avere tutto l'occorrente per poter lavorare.
4)
Dopo aver scelto la tipologia di installazione, ci occorrerà indicare la configurazione del servizio e indicare il percorso di installazione. Per comodità ti consiglio di lasciare le impostazioni di default e proseguire con il tasto Next.






5)
Ti verrà chiesto di decidere di installare Mongo DB Compass. Ti consiglio di installarlo per evitare di farlo in seguito.
6)
Nella schermata finale ti verrà chiesto di iniziare l'installazione e abbiamo concluso questa parte.



-Mongo DB Shell

Mongo DB Shell (o mongosh) è un ambiente di programmazione che ci permette di avere una completa gestione dei nostri database utilizzando dei comandi da terminale.
Per poterlo utilizzare occorre installarlo e lo si può fare dal sito ufficiale [download].
Dopo averlo installato, estrarre l'intera cartella. Una volta estratta, avremo una cartella chiamata mongosh-1.10.1-win32-x64 con all'interno dei file e un'altra directory chiamata bin. Nella cartella bin avremo due file: mongosh.exe che è il file eseguibile e mongosh_crypt_v1.dll.




1)
Questi sono i due file che dovresti avere nella cartella bin che spiegavo qui sopra. Quello che devi fare è di copiare questi due file.









2)
Dopo aver copiato devi seguire questo percorso:
Questo PC / OS P(:C) / Programmi / Mongo DB / Server / 6.0 / bin
Una volta arrivato a questo punto, l'unica cosa che dovrai fare è incollare i file copiati precedentemente.
Per poter proseguire nell'incollatura dei file, è necessario fornire l'autorizzazione cliccando sul tasto Continua.



-Mongo DB Compass

Mongo DB Compass è uno strumento interattivo che ci permette di analizzare e ottimizare i dati. Se nell'installazione di Mongo DB Community Server a accettato di installare anche Mongo DB Compass (A), sei a posto. Altrimenti, non è un problema, andiamo sul sito ufficiale e lo scarichiamo [download].




1)
Dal download sul sito ufficiale avremo questo file eseguibile di cui dovremo solo fare semplicemente il doppio clic.
















2)
In seguito dovremo cliccare su Next per proseguire con l'installazione.



3)
Quando ti si apre questa finestra vuol dire che l'installazione è finita ed è avvenuta con successo e puoi iniziare ad utilizzare Mongo DB Compass.



Comandi

Per poter utilizzare Mongo DB Shell bisogna utilizzare dei comandi nel terminale e per far questo bisogna un minimo saperli. Ecco quali sono!

I nomi in italico sono le parole che bisogna cambiare con quelle che si vogliono

COMANDO FUNZIONE
mongosh Avvia shell di MongoDB (da fare nel proprio terminale)
show dbs Visualizza i propri database
use database Passa dal database corrente a quello desiderato
Se non esiste, ne viene creato uno
db.getName() Ottiene il nome del database corrente
db.createCollection(nome, opzioni) Crea una raccolta con le informazioni contenute tra le parentesi
show collections Mostra tutte le raccolte disponibili
db.getColletionNames() Ottiene il contenuto del database
db.nomeCollection.find() Restituisce tutti i documenti della collection specificata
db.nomeCollection.findOne(filtro) Restituisce il primo documento che soddisfa il filtro specificato
db.nomeCollection.insertOne(documento) Inserisce un nuovo documento nella collection specificata
db.nomeCollection.insertMany(documento 1, documento 2, ...) Inserisce più documenti nella collection specificata
db.nomeCollection.updateOne(filtro, aggiornamento) Aggiorna il primo documento che che soddisfa il filtro specificato con l'aggiornamento
db.nomeCollection.updateMany(filtro, aggiornamento) Aggiorna tutti i documenti che che soddisfano il filtro specificato con l'aggiornamento
db.nomeCollection.deleteOne(filtro) Elimina il primo documento che soddisfa il filtro specificato
db.nomeCollection.deleteMany(filtro) Elimina tutti i documenti che soddisfano il filtro specificato
