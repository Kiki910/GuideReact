Cos'è Java?

Java è un linguaggio di programmazione ad alto livello orientato a oggetti a alla tipizzazione statica. Venne sviluppato da Sun Microsystems (proprietà di oracle Corporation) e venne rilasciato nel 1995. Una caratteristica molto importante di java è la sua portabilità che gli permette di eseguire il codice su diverse piattaforme senza riscriverlo.




-Caratteristiche principali

Le principali caratteristiche di Java sono le seguenti:

  1. orientamento agli oggetti: Java si basa sulla programmazione agli oggetti (OOP) quindi ha un organizzazione in classi e oggetti. Gli oggetti sono istanze di classi e contengono metodi per manipolare i dati;
  2. multi-piattaforma: il codice Java viene compilato in bytecode, un formato di istruzioni eseguibile per una macchina virtuale Java [JVM] e quidni può essere eseguito su qualsiasi dispositivo dotato di una JVM rendendolo un linguaggio WORA [Write Once, Run Anywhere];
  3. robustezza e sicurezza: Java è progettato per prevenire gli errori di runtime comuni, per cui viene eseguita una gestione dell'errore tramite l'eccezioni. Offre anche un modello di sicurezza sandbox che protegge il sistema da codice dannoso;
  4. getionalità automatica della memoria: Java utilizza un meccanismo di gestione della memoria chiamato garbage collection che rilascia la memoria utilizzata dagli oggetti non più referenziati, semplificando la gestione e riducendo i problemi di perdite di memoria;
  5. ampia libreria standard: Java offre molte librerie per svolgere diverse attività di programmazione, come ad esempio per la gestione input/output;
  6. community e supporto: Java è uno dei linguaggi di programmazione più popolari e ha una vasta community di sviluppatori che fornisce supporto, librerie open-source e una grande quantità di risorse di apprendimento.



-Ricerca testuale

Una funzionalità molto importante in Mongo DB è la ricerca testuale che permette di eseguire query in campi stringa per trovare testo o parole specifiche. Una ricerca di testo può essere eseguita con un indice di testo o opertatore $text.
Un indice di testo può essere una stringa oppure un array di elementi e per eseguire la ricerca occorre che la raccolta contenga un indice di testo. Una raccolta può avere un solo indice di testo e più campi possono avere lo stesso indice di testo.
