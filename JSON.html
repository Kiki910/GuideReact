JSON

Cos'è il JSON e come si presenta il suo formato?

Il JSON (JavaScript Objet Notation) è un formato per lo scambio dei dati. È facile da leggere e da scrivere per gli utenti, ma anche facile da interpretare per le macchine. È un sottoinsieme del Linguaggio di Programmazione JavaScript. Possiamo anche dire che il JSON è un formato di testo indipendente dal linguaggio di programmazione, ma utilizza convenzioni simili ai linguaggi del C, C++, Java, JavaScript, Python, ... .


Il JSON si basa su due strutture:

  • insieme di coppie nome:valore, che in altri linguaggi può essere visto come una struct, dizionario...
  • elenco ordinato di valori, come una specie di array, vettore.


Un oggetto è una serie non ordinata di nomi/valori. Un oggetto inizia con '{' [parentesi graffa sinistra] e finisce con '}' [parentesi graffa destra]. Ogni nome è seguito da ':' [due punti] e la coppia di nome/valore sono separata da una ',' [virgola]

Un array è una raccolta ordinata di valori. Un array comincia con '[' [parentesi quadra sinistra] e finisce con ']' [parentesi quadra destra]. I valori sono separati da ',' [virgola].

Un valore può essere una stringa tra virgolette, o un numero, o vero TRUE o falso FALSE o nullo NULL, o un oggetto o un array. Queste strutture possono essere annidate.

Una stringa è una raccolta di zero o più caratteri Unicode tra virgolette; per le sequenze di escape utilizza la barra rovesciata. Un singolo carattere è rappresentato come una stringa di caratteri di lunghezza uno. Una stringa è molto simile ad una stringa C o Java.

Un numero è molto simile ad un numero C o Java, a parte il fatto che i formati ottali e esadecimali non sono utilizzati.

I caratteri di spaziatura possono essere inseriti in mezzo a qualsiasi coppia di token.

La fetch

La Fetch è un metodo che ci permette di mandare una richiesta al server e caricare informazioni ogni volta che ne abbiamo bisogno.

Per usare la fetch servono 3 "parole chiavi":

  • fetch(URL): ha bisogno dell'URL da raggiungere per estrapolare le informazioni;
  • .then(): il primo then() serve per trasformare le informazioni prese tramite l'URL in testo e poi convertirlo in JSON;
  • .then(): il secondo then() serve per prendere i suoi dati.

                        
        const visualizza = () => {

            const URL = "https://raw.githubusercontent.com/icobasco/sample_data_files/master/pera_misure_sample.json"

                        let nuovoElemento = document.createElement("div");
                        nuovoElemento.innerHTML = element.lora_device_id;
                        document.getElementById("device").appendChild(nuovoElemento);
                    })
                })
            }
        }
