Cos'è JavaScript Avanzato?

JavaScript Avanzato è un modulo che ci permette di mettere in atto l'ES6 (ossia EcmaScript, ultima versione di JavaScript).
Per iniziare, bisogna avere un minimo di conoscenza di JavaScript. JavaScript è un linguaggio di programmazione utilizzato nella programmazione Web lato client, per la creazione di effetti dinamici attraverso delle funzioni di script invocate da eventi dall'utente, ad esempio attraverso un click del mouse su un button.

Template String

Funzione map()
La funzione map() è una funzione molto utilizzata in JavaScript perchè permette di ritornare un nuovo array. Praticamente la map() chiama una funzione una volta per ogni elemento in un array, senza modificarne l'arry originale. Molto importante è l'utilizzo di questa funzione quando si utilizza un JSON, ad esempio; questo perché ci permette di ciclare tutto il contenuto dei dati che abbiamo estrapolato dal server.

La destrutturazione
La destrutturazione ci permette di "spacchettare" un array o un oggetto in variabili distinte; in poche parole, possiamo dividere i dati di un array o di un oggetto in varibile separate. 

                        
        const colori = ["rosso", "giallo", "blu", "verde", "bianco", "azzurro", "nero"];

        const [colore1, colore2, colore3] = colori

        console.log(colore1, colore2, colore3);   // come risultato otteniamo: rosso, giallo, blu
